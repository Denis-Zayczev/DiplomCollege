Добро пожаловать!

SQL Server является одной из наиболее популярных систем управления базами данных (СУБД) в мире. Данная СУБД подходит для самых различных проектов: от небольших приложений до больших высоконагруженных проектов.

SQL Server был создан компанией Microsoft. Первая версия вышла в 1987 году. А текущей версией является версия 19, которая вышла в 2019 году.

Данный практикум был разработан на основе SQL server 2008 года и может быть интересен для всех, интересующихся изучением баз данных.

В состав практикума входят:

  • Лабораторные работы;
  • Контроль;
  • Справочный материал по языку SQL;
  • Список источников для самостоятельного изучения.
